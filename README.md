# Bid Builder

A web application for creating and managing bid proposals.

## Features

- Create and manage bid proposals
- Template-based bid generation
- Cost calculation and estimation
- Document export capabilities
- User management and authentication

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone <repository-url>
cd bid_builder
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```
